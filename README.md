一个阅读大藏经的工具。

## 缘起

我喜欢读经，除了自己收藏的纸质经书以外，我也常常在电脑手机上看。[deerpark.app][2] 曾是我的首选，但是几个月前它宣布要关闭了，于是我想寻找一个替代，不过一直没有遇见。与此同时，我也在学习使用AI，用它帮我完成各种工作：写代码，管理服务器，陪练英语等等。最近我感觉我的AI订阅经常有用量剩余，就想不如用来做点什么分享出去吧！正好我想要一个阅藏网站，所以就动手做了这么一个网站。

（PS：做完以后发现 deerpark 又重新上线了。）

## 阅藏

可以通过这个地址访问我的阅藏网站：[cbeta-reader.pages.dev][1] ，它基于最新的「CBETA 電子佛典集成」数据，包含其中的大正新脩大藏經、卍新纂續藏經選錄和太虛大師全書的所有内容。

我只做了最基本的功能：简洁的界面，字体，书签和阅读进度。这些对我来说就很足够。其实在法中有一个概念，叫“薰习”，意思类似于“近朱者赤”，就是说佛陀的教法会像香一样，慢慢使人身染香气，使人清浄。如果我们想用头脑的方式去理解，去研究经中的含义，这样反而会落到世间法的层面，反而有可能对修行造成障碍。“读书百遍，其义自见”，多读，其中的含义是会自然显现的。

## 本地使用

除了在线使用以外，你也可以在你的电脑上本地部署。

### 准备工作

需要安装 [uv](https://docs.astral.sh/uv/)（Python 包管理器）和 Python 3.13+。

```bash
git clone https://github.com/semio/cbeta-reader.git
cd cbeta-reader
uv sync
```

### 获取 CBETA 数据

从 [CBETA 官网](https://www.cbeta.org/) 下载「CBETA 電子佛典集成」，将其中的 `Bookcase/CBETA` 目录放到项目的 `data/` 下，使目录结构为：

```
data/Bookcase/CBETA/
├── catalog.txt
├── XML/
│   ├── T/
│   ├── X/
│   └── ...
└── ...
```

### 方式一：Flask 开发服务器

适合本地浏览，支持实时刷新：

```bash
uv run cbeta-reader
```

然后在浏览器中打开 http://localhost:5000 。

### 方式二：生成静态网站

适合部署到 Cloudflare Pages 等静态托管服务：

```bash
# 生成所有藏经（耗时较长）
make build

# 只生成特定藏经，例如大正藏和卍续藏
make build COLLECTIONS="T X"

# 查看可用的藏经列表
make list
```

生成的静态文件在 `dist/` 目录下。
